# Variables

Replace text in <...> to make the code to the values specific for your project.

1. stackoverflow_test - Looker project and its model file name. We recommend for a Looker project and its model file (if there is one) to have the same name.
2. BigQuery - database name where the data is located and to which is the connection in the model file point to.
3.  `prj-s-dlp-dq-sandbox-0b3c.EK_test_data.stackoverflow_users` - table name that is going to define the 1st view
4. stackoverflow_users - view name tha that you want your 1st view to have
5. `prj-s-dlp-dq-sandbox-0b3c.EK_test_data.stackoverflow_badges` - table name that is going to define the 2nd view
6. <view_name_2> - view name tha that you want your 2nd view to have
7. <explore_name_1> - explore name for your 1st explore
8. <datagroup_name_1> - datagroup name
